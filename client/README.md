# Stock Portfolio API

A Frontend for the RESTful API, built with React, TypeScript, and TanStack Router.

## Tech Stack

- **Runtime:** Node.js
- **Framework:** React
- **Routing:** TanStack Router
- **Language:** TypeScript
- **Package Manager:** pnpm
- **Styling:** Tailwind CSS
- **Icons:** Lucide Icons

## Prerequisites

- Node.js (v18 or higher)
- pnpm
- A running instance of the Stock Portfolio API backend
